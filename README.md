# Projeto POO: RPG Character Creator
Projeto de criador de personagens de RPG para finalização de turma de Programação Orientada a Objetos 2026.

### Participantes e Informações
O projeto foi criado pelos alunos: **Leonardo Souza Leite**, **Mariana dos Santos Cardoso** e **Pedro França Nunes Moreira**.

A ideia do projeto foi de um site para a criação de personagens de RPG usando o paradigma de orientação a objetos. Por meio da criação da classe pai **Personagem**, deu-se origem às classes filhas: *Bárbaro*, *Caçador* e *Mago*, cada qual com suas subclasses. Também seria implementado um sistema de combate, que não foi totalmente concluído a tempo da apresentação.
